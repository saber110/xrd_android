### 开发约定
- 及时沟通
- 用json进行前后端数据交换，采用驼峰命名
- 统一使用git，每天进行pull request
- 后台需对提交的数据进行安全性检查
- 安卓使用Android Studio开发
- 所使用接口前后端要及时沟通
